# TechTrust 🛠️

> **Projeto Integrado Multidisciplinar (PIM III)** > Marketplace de serviços técnicos focado em segurança, confiança e eficiência na conexão entre clientes e profissionais.

---

## 📖 Sobre o Projeto
O **TechTrust** foi desenvolvido como parte do currículo acadêmico de ADS da UNIP. O objetivo é resolver a dificuldade de encontrar profissionais técnicos qualificados, oferecendo uma plataforma onde a transparência e a organização de dados são os pilares centrais.

## 🚀 Funcionalidades Principais
- **Gestão de Usuários:** Cadastro e autenticação de clientes e prestadores.
- **Busca Especializada:** Filtros por categoria de serviço e avaliação.
- **Painel Administrativo:** Visão geral para gestão de chamados.
- **Interface Responsiva:** Design moderno e minimalista focado em UX.

## 🛠️ Tecnologias Utilizadas
Este projeto utiliza um stack focado em performance e organização:
- **Linguagens:** Python (Lógica) e C (Módulos de Performance).
- **Banco de Dados:** Modelagem SQL Relacional para gestão de usuários e serviços.
- **Frontend:** HTML5 e CSS3 com estética minimalista.
- **Deploy:** Vercel / GitHub Pages.

## 🗄️ Estrutura de Banco de Dados
O projeto conta com uma modelagem robusta, focada em:
- Tabelas de Clientes e Técnicos.
- Histórico de Atendimentos e Ratings.
- Categorização de Especialidades.
